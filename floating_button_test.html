Floating Action Button Test

This page tests the floating action button functionality and responsive design.

Test Instructions:

  • Check if the floating button appears in the bottom-right corner
  • Verify the glowing animation effect
  • Test hover effects (shimmer animation)
  • Test click functionality
  • Resize the window to test responsive behavior
  • Test keyboard navigation (Tab to focus, Enter to activate)
  • Test on mobile devices for touch interactions

Expected Behavior:

  • Button should glow with a pulsing green animation
  • On hover, button should lift up and show shimmer effect
  • On mobile, button should be smaller and positioned appropriately
  • Button should be accessible via keyboard navigation
  • Clicking should navigate to developer information page

Responsive Breakpoints Test

Resize your browser window to test these breakpoints:

  • Desktop (>768px): Full size button (180x60px)
  • Tablet (≤768px): Medium button (160x50px)
  • Mobile (≤480px): Small button (140x45px)

Accessibility Test

Test keyboard navigation:

  1. Press Tab multiple times to navigate to the floating button
  2. The button should show a yellow focus outline
  3. Press Enter or Space to activate the button
  4. Screen readers should announce "Contact Developer - Learn more about the developer"
